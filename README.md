# Porker
The best game of <a href="https://itch.io/jam/oink-jam" target="_blank">Oink Jam (2026)</a>.

Porker is a game about playing poker while you lost a big ammount of money.

Discover and play new hands, defeat the enemies and gain all the gold, or lose everything trying...

How long can you keep playing? Will NG+ scare you away? <a href="https://yasushiki.itch.io/porker" target="_blank">Play and find out!</a>
